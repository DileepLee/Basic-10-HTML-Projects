In HTML, paragraphs are defined with the HTML tag. A paragraph element in an HTML document is made up of content — text, images, or other content like form fields — that appears between an opening tag and closing tag. Paragraphs belong to the class of elements called block-level elements The HTML element defines a paragraph.

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.
